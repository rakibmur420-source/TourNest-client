# 📚 TourNest – Travel Booking Platform (Client)

A modern full-stack travel booking platform where travelers can discover exciting destinations, explore tour packages, book trips, and manage their travel experiences through a clean and responsive interface.

## 🌐 Live Demo

https://tour-nest-client-beta.vercel.app/

## 🔗 Server Repository

https://github.com/rakibmur420-source/TourNest-server

---

## ✨ Key Features

- Secure Email & Password Authentication
- Google OAuth Login (Firebase)
- JWT Protected Routes
- Browse and Search Tour Packages
- View Detailed Package Information
- Online Tour Booking System
- Manage Personal Bookings
- Responsive Design (Mobile, Tablet & Desktop)
- Dark & Light Theme
- Image Upload with ImgBB
- Framer Motion Animations
- Toast Notifications
- Role-based Dashboard

## 🔐 Environment Variables

Create a `.env.local` file in the root directory.

```env
VITE_API_URL=your_server_url

VITE_FIREBASE_API_KEY=your_firebase_api_key
VITE_FIREBASE_AUTH_DOMAIN=your_firebase_auth_domain
VITE_FIREBASE_PROJECT_ID=your_project_id
VITE_FIREBASE_STORAGE_BUCKET=your_storage_bucket
VITE_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
VITE_FIREBASE_APP_ID=your_app_id

VITE_IMGBB_API_KEY=your_imgbb_api_key
```

## 🚀 Getting Started

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
